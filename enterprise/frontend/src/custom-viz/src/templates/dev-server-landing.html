Connect to your Metabase instance

  1. In your local Metabase instance go to Admin → Custom visualizations → Development
  2. Set the Dev server URL field to:
  3. Open a question and change its visualization type — your visualization will be available in Custom visualizations section in the sidebar

Changes you make to your source code will automatically reload in Metabase.

How it works

  1. This server serves your built bundle from dist/
  2. Metabase fetches /index.js and assets from this server
  3. An SSE connection at /__sse notifies Metabase when you rebuild
  4. Metabase re-fetches and re-registers your visualization automatically
